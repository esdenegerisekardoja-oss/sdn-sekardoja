    Pengaturan Sinkronisasi Awan (Firebase Real-Time Config)

    Agar semua perubahan yang Anda simpan di satu perangkat (misal HP) langsung tersinkronisasi dan tampil di semua perangkat (misal laptop/perangkat pengunjung lain), Anda dapat memasukkan kunci konfigurasi Firebase mandiri di bawah ini. Jika dikosongkan, website akan secara otomatis mencadangkan data di penyimpanan lokal browser.

    Jika Anda tidak memiliki Firebase config, website akan berjalan pada mode **Penyimpanan Lokal Browser** secara aman.
